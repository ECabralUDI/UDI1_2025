Esta herramienta funciona como un sistema de control de versiones e implementa un __grafo dirigido acíclico__ para llevar el control de los cambios a lo largo de la historia del proyecto.

Este grafo se forma mediante la creacion de nodos (vértices) que apuntan a otros realizados con anterioridad __(aristas)__, para identificar cuáles fueron las acciones realizadas históricamente.
